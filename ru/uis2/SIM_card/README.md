## SIM-карты

<br>
<br>

<img src="SIM_card.svg" alt="" width="100%" height="260px"/>

<br>
<br>

SIM-карты — это FMC-номера, часть корпоративной телефонии UIS. Такой номер позволяет сотруднику использовать обычный мобильный телефон как внутренний номер АТС. То есть с одной, это полноценный виртуальный номер, на который можно настраивать входящие сценарии и использовать все функции АТС. С другой – это обычная SIM-карта, которую сотрудник может использовать для исходящих звонков.

<br>

Заказать SIM-карты можно через вашего менеджера. После того как номера привяжут к лицевому счёту, они появятся в списке, и их можно будет настраивать.

<br>

В настройках доступны:

- **Канальность** — сколько одновременных звонков может принимать номер.
- **Домашний регион** — если включён, при внутренних звонках на SIM-карту отображается короткий номер звонящего. Но важно: при включении этой опции SIM-карта не будет работать за пределами выбранного региона.

<br>
Отдельно настраивается **отображение номера при исходящих звонках** — какой номер увидит клиент, когда ему позвонит сотрудник. Доступны варианты:

- **Фиксированный номер.** При получении звонка клиент увидит фиксированный номер. Сотрудник сможет, при желании, изменить этот номер в софтфоне. Изменение отразится в этом разделе.
- **Знакомый номер.** Клиент увидит знакомый номер, на который он звонил сам или с которого компания звонила ему ранее. Если знакомого номера нет, он увидит номер по умолчанию.
- **Знакомый или региональный**. Клиент увидит знакомый номер, на который он звонил сам или с которого компания звонила ему ранее. Если знакомого номера нет, он увидит номер в коде своего региона. Если такого номера тоже нет, отобразится номер по умолчанию
- **Случайный номер.** Клиент увидит случайный номер из всего списка подключенных номеров компании
- **Случайный региональный.** Клиент увидит случайный номер в коде своего региона. Если такого номера нет, он увидит номер по умолчанию.

<br>

Есть и расширенная настройка — **ограничения по регионам**. Она позволяет разрешать или запрещать использование определённых номеров при звонках в конкретные регионы. Например, для Санкт-Петербурга можно разрешить показывать только номера 812.

<br>

Для отключения SIM-карты обратитесь к своему менеджеру.
