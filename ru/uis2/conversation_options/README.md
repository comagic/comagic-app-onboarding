## Опции разговора

<br>
<br>

<img src="conversation options.svg" alt="" width="100%" height="260px"/>

<br>
<br>

Во время разговора сотрудник может воспользоваться рядом функций Виртуальной АТС. Например, вызвать своего тренера, перевести звонок на другого сотрудника или проставить тег на звонок. Для этого ему нужно просто нажать одну из клавиш на своём телефоне или в приложении.

<br>

В этом разделе можно настроить, какую опцию будет активировать каждая из клавиш.

<br>
<br>

### Доступные опции

<br>

**Проставить тег** – звонку проставляется метка для статистики и анализа. Поставленные теги отобразятся в отчетах и доступны для фильтрации.

<br>

**Выполнить сценарий** – звонок приостанавливается и направляется на обработку по сценарию входящего звонка. Клиент услышит музыку и сообщения, настроенные для этого сценария. У нажавшего клавишу сотрудника звонок сбросится.

<br>

**Назначить персонального менеджера** – номер клиента сохраняется в контактах, а сотрудник, нажавший кнопку, закрепляется за ним в качестве персонального менеджера.

<br>

**Позвонить тренеру** – к разговору подключается тренер, который слышит всех, а клиент не может слышать тренера. Настройка тренеров доступна в разделе <a href="/coach">Тренер</a>.

<br>

**Добавить в черный список** – номер звонящего сохраняется в <a href="/link">черном списке</a>. Дальнейшие звонки с этого номера будут заблокированы.

<br>

**Сохранить контакт** – номер клиента сохраняется в выбранную <a href="/contacts">группу контактов</a>.

<br>

**Включить запись разговора** – включается запись звонка (если она не была активирована по умолчанию). Настройка записи разговоров находится в <a href="//record-settings">соответствующем разделе</a>.

<br>

**Отправить SMS-визитку** – клиенту отправляется <a href="/sms-greeting">SMS-визитка</a> с информацией о компании.

<br>

**Перенести звонок на другое устройство** – активируется дозвон на другие номера сотрудника, позволяя перенести звонок на другое устройство без прерывания разговора.

<br>

**Трансфер** — перевод входящего вызова на другого сотрудника по внутреннему номеру без разрыва соединения с помощью клавиши #. Настройки трансфера доступны в разделе <a href="/transfer">Трансфер звонка</a>.
