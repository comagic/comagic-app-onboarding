## Списки входящих номеров

<br>

<img src="lists_numbers.svg" alt="" width="100%" height="260px"/>

<br>

**Списки номеров** нужны для работы операции «Распределение по спискам номеров» в сценариях входящих звонков. В зависимости от того, входит ли номер клиента в определённый список, виртуальная АТС может применять разные сценарии обработки звонка. Это помогает персонализировать обслуживание и точнее настраивать сценарий звонка.

<br>

<br>

#### В этом разделе вы можете:

<br>

- создавать, редактировать и удалять списки номеров;
- отслеживать, где используется каждый список;
- добавлять комментарии для удобства.

<br>

<br>

#### Создание списка номеров

<br>

1. Укажите название списка, это позволит быстрее находить список в других настройках.
2. Добавьте комментарий (по желанию). Это полезно, если список используется для конкретного сценария.
3. Выберите способ добавления номеров:
- одиночное добавление, где номера вводятся вручную;
- массовое добавление, через загрузку файла в формате .csv (размер до 3 МБ).

<br>

<Alert backgroundColor="#ccc" borderRadius="12" padding>Для одиночного добавления вы можете указать в поле конкретный номер, задать диапазон номеров (с помощью Х на конце) или выбрать номер из существующих контактов.</Alert>

<br>

Все указанные номера будут отображены в таблице. Вы сможете проверить, все ли необходимые номера были добавлены в список, прежде чем сохранить его.

<br>

Один номер может входить в несколько списков номеров. Приоритет использования списка задается непосредственно в операции распределения в сценарии.

<br>

<br>

#### Пример работы

<br>

Есть список «Номера Санкт-Петербурга». В него входят различные диапазоны номеров, в том числе 7921ХХХХХХХ (это префикс мобильного оператора в Петербурге).

<br>

Есть <OnboardingLink to="link">сценарий входящего звонка</OnboardingLink>. В нем создана операция «Распределение по спискам номеров». В ней настроено, что все звонки с номеров из списка «Номера Санкт-Петербурга» направляются в петербуржский отдел продаж.

<br>

В компанию поступил звонок с номера +7(921)333-44-55. Идет проверка: этот номер есть в списке «Номера Санкт-Петербурга»? Да, такой номер подходит под указанный диапазон – звонок поступает менеджерам в Петербурге.
